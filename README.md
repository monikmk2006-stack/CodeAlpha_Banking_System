# CodeAlpha Banking System

## Project Overview

This project is a Banking System developed in C++ as part of the CodeAlpha C++ Programming Internship.

The system allows users to create customer accounts, perform banking operations such as deposits, withdrawals, and fund transfers, and view transaction history.

## Features

* Create Customer Accounts
* Account Management
* Deposit Money
* Withdraw Money
* Fund Transfer Between Accounts
* View Account Balance
* Transaction History Tracking
* File Handling for Data Storage

## Classes Used

### Customer

Stores customer details such as:

* Customer ID
* Customer Name
* Contact Information

### Account

Stores account-related information:

* Account Number
* Customer Details
* Account Balance

### Transaction

Stores transaction details:

* Deposit Records
* Withdrawal Records
* Fund Transfer Records
* Transaction Date and Time

## Functionalities

### Create Customer

Creates a new customer profile and account.

### Deposit

Adds funds to an account and updates the balance.

### Withdraw

Removes funds from an account after checking available balance.

### Transfer Funds

Transfers money between two valid accounts.

### Transaction History

Displays recent account transactions.

### Account Information

Shows:

* Account Number
* Customer Name
* Current Balance
* Transaction Details

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* File Handling
* Standard Template Library (STL)
